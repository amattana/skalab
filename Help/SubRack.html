Subrack


The SKA-Low Subrack is a 6U rack mount EMC shielded case which hosts:


- A Subrack Management Board (SMB)

- 8 TPM boards

- 2 PSU

- A backplane board


The SMB provides access thanks to a CPU running a Debian Linux OS.


The SMB allows basic operations:


- Power ON and Power OFF a TPM board

- Manage the speed of the fans to control the airflow for the heatsinks

- Monitor the temperature with some sensors to the SMA, the backplane, and the TPMs.

- Monitor and limit the current supplied by the PSU.


The SMB provides a gigabit ethernet switch and assign an IP address to each TPM after a Power ON operation.


A python webserver provides the comunication and replies to user queries with JSON strings.
